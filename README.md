# 環境構築

## nodeのインストール

[ここから](http://nodejs.org/) nodeのインストールをする。（設定はデフォルトのままポチるだけでok）

コマンドで以下を入力して、出力を確認する。

``` 
c:¥user¥retro> node -v 
v12.18.1

c:¥user¥retro> npm -v 
6.14.5
```

こんな感じだったらオッケー


# プロジェクトのクローン

このプロジェクトをforkする。
forkした後にそのプロジェクトをcloneする。

コマンドプロンプトでforkしたレポジトリまで移動して以下を入力

```
c:¥user¥retro> cd retro
c:¥user¥retro> npm start
```

ローカルサーバが立ち上がることを確認する。

以上！
